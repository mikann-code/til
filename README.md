# TIL (Today I Learned)

インターンで得た知見やメモを記録する場所。

## 構成

```
til/
└── dev/
    ├── frontend/
    │   ├── nextjs/
    │   └── javascript/
    ├── backend/
    │   ├── rails/
    │   ├── ruby/
    │   └── database/
    ├── git/
    └── team/
```

## カテゴリ

### frontend
| フォルダ | 内容 |
|---|---|
| `nextjs/` | App Router、SSR/SSG、コンポーネントなど |
| `javascript/` | JS/TS全般 |

### backend
| フォルダ | 内容 |
|---|---|
| `rails/` | ルーティング、ActiveRecord、Gemなど |
| `ruby/` | Ruby言語仕様全般 |
| `database/` | SQL、マイグレーション、インデックスなど |

### その他
| フォルダ | 内容 |
|---|---|
| `git/` | コマンド、ブランチ戦略など |
| `team/` | コードレビューの作法、開発フローなど |
